AES加密协处理器

C MCS-51
MCS-51
发布日期:   2022-03-01

0.前言

目标一

使用Verilog语言实现AES加密协处理器：
（1）使用Verilog完成XTEA/AES/SM4/MD5/PRESENT/SHA-1基本模块；
（2）设计一个状态机，从SRAM中取加密的明文数据，然后交由密码算法加密引擎进行加密，而后状态机从加密算法引擎中取得加密结果并写入SRAM中。

目标二

使用C语言实现XTEA/AES/SM4/MD5/PRESENT/SHA-1加密算法：
（1）基于任意的C语言开发平台，实现XTEA/AES/SM4/MD5/PRESENT/SHA-1加密算法；
（2）评估 C 语言运行时间。

目标三

对比分析软件、硬件实现的加密效率。
（1）推断出所设计XTEA/AES/SM4/MD5/PRESENT/SHA-1硬件的最高频率，得出硬件实现的加密速率；
（2）得出软件实现的加密速率；
（3）对比分析软硬件实现速率。

1.C语言实现

主函数中定义声明了明文密文秘钥，malloc函数申请为扩展秘钥w预先扩展，使用秘钥扩展函数进行秘钥扩展后，使用扩展秘钥与明文通过加密函数生成密文，最终使用printf输出。

1.2 秘钥扩展模块

void aes_key_expansion(uint8_t *key, uint8_t *w) &#123;

    uint8_t tmp[4];
    uint8_t i;
    uint8_t len = 4*11;

    for (i = 0; i < 4; i++) &#123;

        w[4*i+1] = key[4*i+1];
        w[4*i+2] = key[4*i+2];
        w[4*i+3] = key[4*i+3];
    &#125;

    for (i = 4; i < len; i++) &#123;
        tmp[0] = w[4*(i-1)+0];
        tmp[1] = w[4*(i-1)+1];
        tmp[2] = w[4*(i-1)+2];
        tmp[3] = w[4*(i-1)+3];

        if (i%4 == 0) &#123;

            rot_word(tmp);
            sub_word(tmp);
            coef_add(tmp, Rcon(i/4), tmp);

        &#125; 

        w[4*i+0] = w[4*(i-4)+0]^tmp[0];
        w[4*i+1] = w[4*(i-4)+1]^tmp[1];
        w[4*i+2] = w[4*(i-4)+2]^tmp[2];
        w[4*i+3] = w[4*(i-4)+3]^tmp[3];
    &#125;
&#125;

秘钥扩展函数输入为一个128位秘钥，输出一个4448位的秘钥矩阵（主函数中w使用malloc预先申请44*4的字节空间），函数中使用到了临时变量tem存储每一次循环生成的临时参数，i用作循环计数。将原始秘钥放入扩展矩阵w的低四列，对于其他四十列，通过判断列数是否为四的倍数，若为4的倍数，则先进行循环左移一位，s盒字节替换，轮常量异或。若不是4的倍数，则直接进行与前4位进行异或。

1.4 核心加密模块

void aes_cipher(uint8_t *in, uint8_t *out, uint8_t *w) &#123;

    uint8_t state[4*Nc];
    uint8_t r, i, j;
    //行列转换
    for (i = 0; i < 4; i++) &#123;
        for (j = 0; j < Nc; j++) &#123;
            state[4*i+j] = in[i+4*j];
        &#125;
    &#125;
    add_round_key(state, w, 0);

    for (r = 1; r < 10; r++) &#123;
        sub_bytes(state);//字节替换
        shift_rows(state);//行移位
        mix_columns(state);//列混合
        add_round_key(state, w, r);//轮秘钥加
    &#125;

    sub_bytes(state);//字节替换
    shift_rows(state);//行移位
    add_round_key(state, w, 10);//轮秘钥加

    for (i = 0; i < 4; i++) &#123;
        for (j = 0; j < Nc; j++) &#123;
            out[i+4*j] = state[Nc*i+j];
        &#125;
    &#125;
&#125;

先将明文进行行列转换，转化为列矩阵，进行第一轮轮秘钥加，然后进行九轮循环字节替换、行移位、列混合、轮秘钥加，最后再循环一次，不包含列混合，最后行列转换将密文转换为行矩阵输出。

1.5 轮秘钥加

void add_round_key(uint8_t *state, uint8_t *w, uint8_t r) &#123;
    
    uint8_t c;
    
    for (c = 0; c < Nc; c++) &#123;
        state[Nc*0+c] = state[Nc*0+c]^w[4*Nc*r+4*c+0];   
        state[Nc*1+c] = state[Nc*1+c]^w[4*Nc*r+4*c+1];
        state[Nc*2+c] = state[Nc*2+c]^w[4*Nc*r+4*c+2];
        state[Nc*3+c] = state[Nc*3+c]^w[4*Nc*r+4*c+3];  
    &#125;
&#125;

Nc*n+c代表第c行第n列的数，将状态矩阵与扩展秘钥对应的矩阵进行异或，结果存入临时矩阵继续进行循环。

1.7 行移位模块

void shift_rows(uint8_t *state) &#123;

    uint8_t i, k, s, tmp;

    for (i = 1; i < 4; i++) &#123;
        s = 0;
        while (s < i) &#123;
            tmp = state[Nc*i+0];
            
            for (k = 1; k < Nc; k++) &#123;
                state[Nc*i+k-1] = state[Nc*i+k];
            &#125;
            state[Nc*i+Nc-1] = tmp;
            s++;
        &#125;
    &#125;
&#125;

把状态矩阵中复制出来，然后对它们行进左移相应的位数，第0行不移动，第1行循环左移1字节，第2行循环左移2字节，第3行循环左移3字节。然后再复制回去状态矩阵中

void coef_mult(uint8_t *a, uint8_t *b, uint8_t *d) &#123;

    d[0] = gmult(a[0],b[0])^gmult(a[3],b[1])^gmult(a[2],b[2])^gmult(a[1],b[3]);
    d[1] = gmult(a[1],b[0])^gmult(a[0],b[1])^gmult(a[3],b[2])^gmult(a[2],b[3]);
    d[2] = gmult(a[2],b[0])^gmult(a[1],b[1])^gmult(a[0],b[2])^gmult(a[3],b[3]);
    d[3] = gmult(a[3],b[0])^gmult(a[2],b[1])^gmult(a[1],b[2])^gmult(a[0],b[3]);

        if(b < 128) gmultres=b<<1;
        else gmultres=(b<<1)^0b00011011;
    &#125;
    else &#123;
